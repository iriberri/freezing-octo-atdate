freezing-octo-atdate
====================

Based on the stream socket client demo contained in Beej's Guide to Network Programming.

Usage:
```
atdate [-h serverhost] [-p port] [-m cu | ct | s ] [-d]


* -h serverhost: TIME server 
* -p port: default port is 37.
* -m mode:
  * cu: UDP client
  * ct: TCP client
  * s: server
* -d: debug mode 
```
